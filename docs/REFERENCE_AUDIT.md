# Reference audit

The bibliography distinguishes imported classical facts, earlier tetrahedral
tiling classifications, the T2 self-prior-art source of the open problem and
figures, and results proved in the present paper.

The public companion repository makes no blanket “first ever” claim. In
particular, older congruent-triangle/tetrahedron literature and the
Rosenblatt–Shapiro paper remain mandatory scope checks. Because legal full
text is unavailable, novelty language remains at the narrower
object-and-theorem level supported by source-located evidence.

Figure provenance is separately recorded in `THIRD_PARTY_NOTICES.md`.
Bibliographic metadata and claim-level novelty should be rechecked immediately
before submission or archival release.
